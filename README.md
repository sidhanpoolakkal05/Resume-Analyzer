# AI Resume Analyzer

AI-powered resume analysis tool that compares resumes with job descriptions and generates match scores, skill gaps, and improvement suggestions.

## Features

- Resume PDF parsing
- Skill extraction
- Job description matching
- ATS-style scoring
- AI feedback system
- Interactive dashboard
- Skill analytics charts

## Tech Stack

Backend
- FastAPI
- Scikit-learn
- PDFMiner

Frontend
- React
- TailwindCSS
- Recharts

## Installation

### Backend

cd backend
pip install -r requirements.txt
uvicorn main:app --reload

### Frontend

cd frontend
npm install
npm run dev
